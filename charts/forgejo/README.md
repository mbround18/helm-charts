# Forgejo Helm Chart

This Helm chart deploys Forgejo, a self-hosted lightweight software forge, on a Kubernetes cluster.

## Installation

To install the chart:

```bash
helm repo add my-repo https://mbround18.github.io/helm-charts # Or your custom repo
helm install my-forgejo my-repo/forgejo --version 0.1.4 # Use the correct version
```

## Configuration

The following table lists the configurable parameters of the Forgejo chart and their default values.

| Key | Type | Default | Description |
|---|---|---|---|
| `replicaCount` | int | `1` | Number of Forgejo instances to deploy. |
| `image.repository` | string | `codeberg.org/forgejo/forgejo` | Forgejo image repository. |
| `image.tag` | string | `15` | Forgejo image tag. |
| `image.pullPolicy` | string | `IfNotPresent` | Image pull policy. |
| `service.http.port` | int | `3000` | HTTP port for Forgejo. |
| `service.ssh.port` | int | `22` | SSH port for Forgejo. |
| `ingress.enabled` | bool | `false` | Enable/disable Ingress. |
| `ingress.hosts` | list | `[{host: forgejo.local}]` | List of Ingress hosts. |
| `persistence.enabled` | bool | `true` | Enable/disable persistence for `/data`. |
| `persistence.size` | string | `10Gi` | Size of the persistent volume. |
| `persistence.storageClass` | string | `""` | Storage class for the persistent volume. |
| `postgresql.enabled` | bool | `true` | Enable/disable the bundled PostgreSQL dependency. |
| `externalSecrets.enabled` | bool | `true` | Enable/disable ExternalSecrets integration. |
| `externalSecrets.secretStore` | string | `vault-kv` | Name of the ClusterSecretStore or SecretStore. |
| `externalSecrets.secretName` | string | `forgejo-external-secrets` | Name of the Kubernetes Secret generated by ExternalSecret. |
| `externalSecrets.keys.databasePassword` | string | `"forgejo/data/databasePassword"` | Remote key for the database password. |
| `externalSecrets.keys.runnerToken` | string | `"forgejo/data/runnerToken"` | Remote key for the runner registration token. |
| `externalSecrets.keys.githubToken` | string | `"forgejo/data/githubToken"` | Remote key for the GitHub token for mirror cron job. |
| `externalSecrets.keys.forgejoToken` | string | `"forgejo/data/forgejoToken"` | Remote key for the Forgejo token for mirror cron job. |
| `config.database.db_type` | string | `postgres` | Database type. |
| `config.database.host` | string | `postgresql` | Database host. |
| `config.database.name` | string | `forgejo` | Database name. |
| `config.database.user` | string | `forgejo` | Database user. |
| `config.server.domain` | string | `forgejo.local` | Server domain. |
| `config.server.ssh_domain` | string | `forgejo.local` | SSH domain. |
| `config.server.root_url` | string | `http://forgejo.local/` | Root URL for Forgejo. |
| `config.server.http_port` | string | `3000` | HTTP port for Forgejo instance. |
| `runner.enabled` | bool | `true` | Enable/disable the bundled Forgejo runner. |
| `runner.dockerSocket.mount` | bool | `true` | Mount the Docker socket into the runner container. **Security Warning: Review documentation.** |
| `mirrorCronJob.enabled` | bool | `false` | Enable/disable the GitHub to Forgejo mirror cron job. |
| `mirrorCronJob.schedule` | string | `"0 * * * *"` | Cron schedule for the mirror job. |
| `mirrorCronJob.image.repository` | string | `ghcr.io/patnei/github2forgejo` | Mirror cron job image repository. |
| `mirrorCronJob.github_user_or_org` | string | `""` | GitHub user or organization for mirroring. |
| `mirrorCronJob.forgejo_user` | string | `""` | Forgejo user for mirroring. |

Specify each parameter using the `--set key=value[,key=value]` argument to `helm install`. For example:

```bash
helm install my-forgejo my-repo/forgejo 
  --set ingress.enabled=true 
  --set ingress.hosts[0].host=my.forgejo.example.com 
  --set externalSecrets.enabled=true 
  --set externalSecrets.secretStore=my-cluster-secret-store
```

Alternatively, a YAML file that specifies the values for the parameters can be provided:

```bash
helm install my-forgejo my-repo/forgejo -f values.yaml
```
